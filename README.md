# dlspree

镜像内容及版本如下：

+ `cuda`：`10.1`
+ `CUDNN`：`7`
+ `Anaconda`：`2020.02-Linux-x86_64`
+ `openssh-server`
+ `tensorflow-gpu`：`1.15`
+ `tensorboardX`：`1.9`
+ `pytorch`:`1.7`
+ `keras`
+ `autopep8`
+ `torchsnooper`
+ `pysnooper`
+ `NNI`：`1.4`
+ `jupyter_contrib_nbextensions` 
+ `xgboost`
+ `pyecharts`
+ `DGL`
